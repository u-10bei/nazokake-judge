# なぞかけ品質評価へのご協力のお願い

## このタスクについて

2 つの作品（**A** と **B**）を読み比べて、**より優れていると感じた方**を選んでいただきます。それを何度か繰り返すだけの、シンプルなタスクです。

- **所要時間**: 約 25〜35 分
- **必要なもの**: お配りした URL のみ（**アカウント登録は不要**）
- **推奨環境**: スマートフォン（PC でも問題ありません）
- **言語**: 日本語

## 大切なこと

**優劣に「唯一の正解」はありません。** あなたが読んで感じたとおりに、直感で選んでください。深く悩む必要はありません。あなたの感じ方そのものが、この調査で知りたいことです。

---

## 進め方

### 1. URL を開く

お配りした URL をブラウザで開いてください。すぐに説明画面が表示されます。**あなた専用の URL** なので、他の方と共有しないでください。

### 2. 練習（数問）

いきなり本番には入りません。まず数問の**練習**があります。

> 練習中は画面上部に「**練習中（この回答は集計されません）**」と表示されます。操作に慣れるためのものなので、気楽にお試しください。

### 3. 本番の判定（およそ 40 ペア）

「**どちらが優れていますか？**」という画面で、A と B の作品が表示されます。

1. どちらか一方を選ぶ（カードをタップ）
2. 「**送信**」を押す

- **選ぶまで「送信」は押せません**（誤送信を防ぐためです）
- 画面には「**本番 ○ / ○**」と**進捗**が出ます。残りの見通しを持ちながら進められます
- 回答は**1 問ごとにその都度保存**されます

### 4. 作品の評価（Likert）

途中で「**この作品の出来を評価してください**」という画面が出ます。1 つの作品について、画面に示される尺度（例: 1〜7、低い〜高い）で評価してください。

### 5. 事後アンケート

判定が終わったら、最後に短いアンケートにご回答ください。分析のために使わせていただきます。

| 設問 | 回答形式 |
|---|---|
| なぞかけ経験 | ほとんどない / ときどき触れる / よく親しんでいる |
| 自己申告の熟達度 | 1〜5 から選択 |
| 判定時に重視した観点 | 自由記述（例: 意外性、掛詞の巧みさ） |
| 年代 | 10 代〜60 代以上から選択 |

> **なぞかけの経験がなくても大丈夫です。** 経験の有無で回答の価値は変わりません。むしろ様々な方の感じ方を知ることが目的なので、ありのままをお答えください。

### 6. 完了

「**ご協力ありがとうございました**」と表示されたら終了です。ページは閉じていただいて構いません。

---

## 途中でやめても大丈夫です

**中断できます。** 通信が切れたり、用事ができて閉じてしまっても問題ありません。

**同じ URL をもう一度開けば、続きから再開できます。** それまでの回答は保存されており、答えた問題がもう一度出ることはありません。最初からやり直しにはなりませんので、ご安心ください。

---

## 困ったときは

| 表示・症状 | 意味と対処 |
|---|---|
| **「アクセスできません」** | URL が正しく開けていない可能性があります。お配りした URL を、途中で切れていないか確認して開き直してください。それでも駄目な場合はご連絡ください |
| **「完了済み」と表示される** | そのURLでの回答はすべて完了しています。追加の操作は不要です |
| **送信が失敗した / 通信が切れた** | もう一度送信してください。**二重に登録されることはありません**。それでも進まない場合は、一度閉じて同じ URL を開き直すと続きから再開できます |
| **画面が真っ白・進まない** | 通信環境の良い場所で、同じ URL を開き直してください。回答は保存されています |

---

## プライバシーについて

- **お名前・メールアドレスなど、個人を特定する情報は一切お聞きしません。** お答えいただくのは、上記の判定・評価とアンケートのみです。
- 回答はお配りした URL（トークン）に紐づいて記録されます。**通信はすべて暗号化（HTTPS）** されています。
- ご回答は統計的に集計され、**研究目的にのみ**利用します。
